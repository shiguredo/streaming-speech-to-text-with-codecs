# Download whisper.cpp models

Hugging faceからwhisper.cppのモデルをダウンロードするためのスクリプトです。`wget`が必要です。

## 使い方

```bash
./download-ggml-model.sh model_name
```

## 利用可能なモデル名

https://huggingface.co/datasets/ggerganov/whisper.cpp

- tiny
- tiny.en
- base
- base.en
- small
- small.en
- medium
- medium.en
- large
- large-v1
